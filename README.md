# PratiqueReseauNeurone

Ce répertoire contient tous les exercices pratiques sur les **réseaux de neurones**, réalisés sans programmation orientée objet.  

Les exercices sont classés selon leur contexte :  
- **En classe (TD / cours)** : exercices réalisés pendant les séances, permettant de comprendre les concepts de base et de mettre en pratique les notions vues en cours.  
- **À la maison (exercices d’entraînement)** : exercices supplémentaires pour s’entraîner et consolider les compétences.

Ce répertoire sert de **support pratique** pour appliquer les notions théoriques des réseaux de neurones, et peut être utilisé pour révision ou apprentissage autonome.
